# SAIEL OMNI APP

Aplicación Next.js con integración OpenAI.

## Estructura

- Next.js 14
- API Route personalizada
- Núcleo configurable en lib/saielCore.js

## Requisitos

Agregar variable de entorno en Vercel:

OPENAI_API_KEY

## Desarrollo local

npm install
npm run dev

## Producción

npm run build
npm start
